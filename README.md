# clipforge
ClipForgeAI is a production-grade, local AI multi-agent desktop application that automatically transforms long-form videos and YouTube URLs into viral, short-form clips (9:16) tailored for TikTok, YouTube Shorts, and Instagram Reels. Built using PySide6, FFmpeg, faster-whisper , and Google Gemini 1.5/3.5 Flash, it runs fully locally on consumerGPUs
